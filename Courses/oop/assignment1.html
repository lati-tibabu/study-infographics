BINARY

A base-2 numbering system using only two digits: 0 and 1. It is the fundamental, native language of all digital computers.

Examples:

Decimal 13 = 1101 (Binary) Decimal 5 = 101 (Binary)
0-7

OCTAL

A base-8 numbering system using digits 0 through 7. It was historically used as a compact representation of binary data.

Examples:

Decimal 15 = 17 (Octal) Decimal 9 = 11 (Octal)
0-F

HEXADECIMAL

A base-16 numbering system using digits 0-9 and letters A-F. Used for web color codes (e.g., `#FF0000`) and memory addresses.

Examples:

Decimal 255 = FF (Hexadecimal) Web Color White = #FFFFFF (Hexadecimal)

ASCII

(American Standard Code for Information Interchange). A 7-bit character encoding primarily for the English alphabet, digits, and control codes. It is a subset of Unicode.

Examples:

The letter 'A' is represented by decimal 65. The Tab control code is represented by decimal 9.

METHOD (OOP)

A function that is associated with an object or class. It defines the behavior or actions an object can perform.

Examples:

If Dog is a class, dog.bark() is a method (Python). If Car is a class, car.startEngine() is a method (Java).
